    Java容器1——概述

    编程语言 > Java Java 容器
    发布时间 : 2023-07-25 09:28
    字数:131 阅读 :
    评论:

    Java容器主要分为两大类：

    • Collection：一个独立元素的序列，一次存一个元素，是单列集合。Collection又主要包括以下三种：
      • List：存储的元素有序、可重复，是个常用的容器
      • Set：存储的元素无序、不可重复，用于数据需要唯一存在的场景
      • Queue：存储的元素有序、可重复，可指定存取数据的先后顺序
    • Map：一组成对的“键值对”对象，一次存一对元素，是双列集合
